Privacy Policy

Last updated: February 2026

Overview

This service ("Astro-ph Digest API") provides summaries of publicly available scientific papers listed on arXiv.org.

Data Collection

This service does not collect personal information from users.

When requests are made to the API, basic technical information such as IP address, request time, and server logs may be temporarily recorded for operational and security purposes.

Use of Data

Any collected technical information is used solely to:

  • Operate and maintain the API
  • Prevent abuse or misuse
  • Monitor service performance

Third-Party Data Sources

This service retrieves publicly available metadata from arXiv.org. No user data is shared with arXiv or other third parties.

Data Sharing

No personal data is sold, rented, or shared with advertisers or third parties.

Data Retention

Server logs, if collected, are retained only as long as necessary for operational monitoring.
